Всего в API 3 метода:
GET "/" - возвращает статус сервера.
GET "/?text" - возвращает по промту в query params результат работы модели.
POST "/predict" - возвращает по промту в body ответа результат работы модели.

Для запуска API нужно:
1. Скачать все зависимости из файла requirements.txt,
2. В терминале выполнить команду uvicorn main:app --reload

Для запуска пользовательского интерфейса нужно:
1. Скачать все зависимости из файла requirements.txt,
2. В терминале выполнить команду streamlit run main_page.py
